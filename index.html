今天学了git 先是学了时光机穿梭，其中有版本回退：$ git reset --hard commit_id，工作区和暂存区,管理修改：$ git add 后再 $ git commit，撤销修改：$ git checkout -- file，删除修改：$ git rm；然后学了远程仓库，其中有添加远程库：$ git remote add origin git@server-name:path/repo-name.git，git push -u origin master，远程克隆：$ git clone；然后是分支管理，其中有创建与合并分支：$ git checkout -b 和$ git merge ，解决冲突：当Git无法自动合并分支时，就必须首先解决冲突。解决冲突后，再提交，合并完成，分支管理策略和多人协作。
